To run locally:
- install Python 3.14+
- (optional but recommended) create a virtual environment
- install required packages ```python-dotnenv, json, openpyxl, pandas, requests```
- copy ```.env.EXAMPLE``` to ```.env```
- input baselinker API token in ```BASELINKER_TOKEN``` field in ```.env```
- run ```python main.py``` in terminal

Output files will be saved in ```/responses``` directory under ```clean/``` or ```raw/``` accordingly. Final output will be in terminal to be copied.
